# Prueba Técnica - Analista Programador

## Tecnologías utilizadas
- Java Spring Boot
- MySQL
- Angular
- Postman

## Funcionalidad
La aplicación permite:
- Autenticación con Bearer Token
- Consulta de disponibilidad de retiro
- Ejecución de retiro
- Almacenamiento de transacciones en MySQL
- Consulta de reportes

## Endpoints
- POST /api/login
- POST /api/consulta
- POST /api/pago
- GET /api/reportes
